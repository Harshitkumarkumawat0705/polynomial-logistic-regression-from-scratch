# polynomial-logistic-regression-from-scratch
Polynomial Logistic Regression implemented from scratch using NumPy with L2 regularization and non-linear decision boundary visualization.

This project implements Polynomial Logistic Regression from scratch using NumPy. 
The model uses polynomial feature mapping to create a non-linear decision boundary 
and L2 regularization to prevent overfitting.

## Features
- Polynomial feature mapping
- Feature normalization
- Sigmoid function
- Log-loss cost function
- L2 Regularization
- Gradient Descent optimization
- Non-linear decision boundary visualization

## Libraries Used
- NumPy
- Matplotlib
- Scikit-learn (only for dataset generation)

## How to Run
1. Install required libraries:
   pip install numpy matplotlib scikit-learn

2. Run the script:
   python polynomial_logistic_regression.py
